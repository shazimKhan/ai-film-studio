# Village Shopkeeper 01

## Workflow

1. Generate `prompts/FRONT.md`.
2. Approve the identity.
3. Save the approved result as `references/front.png`.
4. Use the approved front image as the identity reference for later angles.
5. Generate the remaining angle prompts one by one.
6. Save each approved image inside `references/`.
