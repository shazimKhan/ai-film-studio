# Prop Bible

The prop bible stores reusable object references for production continuity.

## Required Files

- `prop.yaml`
- `prompt.md`

## Required Folders

- `references/`

Props do not receive placeholder images. Generate or approve prop references outside
the repository, then save them into the prop `references/` folder.

Keep prop names stable across scenes and avoid replacing approved references without a
versioned production decision.
