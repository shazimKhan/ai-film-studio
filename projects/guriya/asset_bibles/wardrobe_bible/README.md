# Wardrobe Bible

Wardrobe references live inside each character folder under `wardrobe/`.

## Required Wardrobe Slots

- `default.png`
- `dress_01.png`
- `dress_02.png`
- `winter.png`
- `wedding.png`
- `sleep.png`

Wardrobe images must be generated or approved manually. Metadata should not invent
fabric, color, fit, or condition before reference images exist.

Use wardrobe references to preserve continuity across prompt compilation, video
generation, and review.
