# Character Prompt Template

Use this reusable template for manually generating character reference images.

```text
Create a production reference image for character asset {character_id}.
Use only approved source material and written continuity notes.
Required view or state: {reference_slot}.
Output must be clean, consistent, and suitable for future identity locking.
Do not add text, logos, watermarks, or unrelated props.
Do not invent unapproved appearance details.
```
