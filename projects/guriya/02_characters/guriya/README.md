# Guriya Character Pack

This pack is the production-facing character reference for Guriya.

Use `MASTER_PROMPT.md` for the identity and wardrobe lock. Use `NEGATIVE_PROMPT.md`
for identity, period, and style exclusions. YAML files store structured policies
for continuity, camera, motion, style, and voice.

Approved HD image references belong in:

```text
projects/guriya/characters/guriya/references/production/
```

Dialogue remains in `10_voiceover/`, not in video prompts.
