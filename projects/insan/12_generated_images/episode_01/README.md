# Episode 01 Generated Images

Store only manually generated and approved Episode 01 images here.

Use:

- `assets/` for approved reusable master assets.
- `shots/` for approved final shot keyframes.
- `_drafts/` folders for rough generations.
- `_rejected/` folders for failed generations.

Do not save fake placeholder PNG files.
Do not generate Qur'anic Arabic text inside images.
Do not save any image that violates the depiction rules.
