# Insan

`Insan` is the primary research-first production project for humanity and Islamic
history. The project uses Qur'an, authentic Hadith, validated Tafsir, Seerah,
classical histories, scholarly interpretation, and modern academic references with
clear source hierarchy.

No episode, scene, shot, prompt, or depiction should enter production until its
factual outline is supported by approved source records.

This foundation intentionally contains no invented episode narrative and no fake
citations.
